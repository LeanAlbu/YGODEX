# YGODEX

YGODEX é um projeto para gerenciamento, visualização e análise de cartas do universo Yu-Gi-Oh!. 

## Funcionalidades
- No momento sequer mostra as cartas direito, mas vamos evoluindo com o tempo...

## Tecnologias Utilizadas
- JavaScript
- HTML + CSS
- YGOPRO API
